I built an AI-powered Email Spam Detection Web Application from scratch using React, Django REST Framework, and Logistic Regression.
🔹 Features
📧 Detects whether an email is Spam or Ham
🧹 Text preprocessing using NLTK
📊 TF-IDF Vectorization
🤖 Logistic Regression for classification
🌐 Django REST API backend
⚛️ React frontend with a modern responsive UI ( mostlty created by chatgpt)
📈 Confidence score for every prediction
🛠️ Tech Stack
React
Django REST Framework
Python
Scikit-learn
NLTK
TF-IDF Vectorizer
Logistic Regression
Tailwind CSS
This project helped me understand the complete machine learning workflow—from data preprocessing and model training to backend API development and frontend integration.
There were many debugging challenges along the way, including model loading, preprocessing, virtual environments, and API integration, but solving those problems taught me just as much as building the model itself.
This is my first end-to-end AI web application, and I'm excited to keep building more machine learning projects.
