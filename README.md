# 📧 AI-Powered Email Spam Detection Web Application

An end-to-end **Machine Learning web application** that classifies emails as **Spam** or **Ham (Not Spam)** using **Logistic Regression**. The project combines a **React frontend**, **Django REST Framework backend**, and a **Scikit-learn machine learning model** to provide real-time spam detection with confidence scores.

---

## 🚀 Features

- 📧 Detects whether an email is **Spam** or **Ham**
- 🧹 Text preprocessing using **NLTK**
- 📊 TF-IDF Vectorization for feature extraction
- 🤖 Logistic Regression for spam classification
- 📈 Displays prediction confidence score
- 🌐 RESTful API built with Django REST Framework
- ⚛️ Modern and responsive React frontend
- 🎨 Styled using Tailwind CSS

## ⚙️ How It Works

1. User enters an email message.
2. The React frontend sends the text to the Django REST API.
3. The backend preprocesses the text using **NLTK**.
4. The saved **TF-IDF Vectorizer** converts the text into numerical features.
5. The trained **Logistic Regression** model predicts whether the email is **Spam** or **Ham**.
6. The prediction and confidence score are returned to the frontend and displayed to the user.
